# statistical-modeling
This code demonstrates a complete statistical modeling workflow in R using:

- tidyverse
- ggplot2
- dplyr
- broom (for model summaries)
- reproducible experiment design

The code includes:
- Synthetic dataset generation
- Exploratory data analysis (EDA)
- A linear regression model predicting `Productivity` from `Experience` and `TrainingHours`
- Visualizations using ggplot2
- Model diagnostics
